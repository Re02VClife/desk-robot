# 机器狗项目 (Dog Robot)

这是一个基于ESP32S3和4个SG90舵机的智能机器狗项目，集成了小智机器人的语音控制功能。

## 硬件需求

- ESP32S3开发板
- 4个SG90舵机
- 麦克风（用于语音输入）
- 扬声器（用于语音输出）
- 电源（推荐5V 2A）

## 硬件连接

| 舵机 | ESP32S3引脚 | 功能 |
|------|------------|------|
| 前左腿 | GPIO12 | 控制前左腿运动 |
| 前右腿 | GPIO13 | 控制前右腿运动 |
| 后左腿 | GPIO14 | 控制后左腿运动 |
| 后右腿 | GPIO15 | 控制后右腿运动 |

## 功能特性

- 🎤 语音控制：通过小智机器人的语音唤醒和对话功能控制机器狗
- 🤖 预设动作：站立、行走、左转、右转、挥手等
- 📱 MCP工具：通过大语言模型API控制机器狗
- ⚙️ 舵机校准：支持设置舵机偏移量进行校准

## 语音控制示例

```
你好小智，让机器狗站起来
你好小智，让机器狗走5步
你好小智，让机器狗向左转3步
你好小智，让机器狗向右转3步
你好小智，让机器狗挥手
```

## MCP工具接口

### 1. `self.dog_robot.stand`
- **功能**：让机器狗站立
- **参数**：无
- **返回**：`true`（成功）

### 2. `self.dog_robot.walk`
- **功能**：让机器狗行走
- **参数**：
  - `steps`：步数（1-20），默认5
  - `speed`：速度（50-500ms），默认200
- **返回**：`true`（成功）

### 3. `self.dog_robot.turn_left`
- **功能**：让机器狗向左转
- **参数**：
  - `steps`：步数（1-10），默认3
  - `speed`：速度（50-500ms），默认200
- **返回**：`true`（成功）

### 4. `self.dog_robot.turn_right`
- **功能**：让机器狗向右转
- **参数**：
  - `steps`：步数（1-10），默认3
  - `speed`：速度（50-500ms），默认200
- **返回**：`true`（成功）

### 5. `self.dog_robot.wave`
- **功能**：让机器狗挥手
- **参数**：无
- **返回**：`true`（成功）

### 6. `self.dog_robot.move_servo`
- **功能**：单独控制一个舵机
- **参数**：
  - `servo`：舵机编号（0-3），0=前左, 1=前右, 2=后左, 3=后右
  - `position`：位置（0-180度），默认90
- **返回**：`true`（成功）

### 7. `self.dog_robot.set_trim`
- **功能**：设置舵机偏移量
- **参数**：
  - `front_left`：前左腿偏移（-20到20），默认0
  - `front_right`：前右腿偏移（-20到20），默认0
  - `back_left`：后左腿偏移（-20到20），默认0
  - `back_right`：后右腿偏移（-20到20），默认0
- **返回**：`true`（成功）

## 编译与烧录

1. **配置板型**：在ESP-IDF配置中选择 `BOARD_TYPE_DOG_ROBOT`
2. **编译固件**：
   ```bash
   idf.py build
   ```
3. **烧录固件**：
   ```bash
   idf.py flash
   ```
4. **监控输出**：
   ```bash
   idf.py monitor
   ```

## 校准指南

1. **站立校准**：
   - 让机器狗执行站立动作
   - 观察机器狗的姿态
   - 使用 `set_trim` 工具调整各个舵机的偏移量
   - 重复上述步骤直到机器狗能够稳定站立

2. **行走校准**：
   - 让机器狗执行行走动作
   - 观察行走姿态和方向
   - 调整相应舵机的偏移量
   - 重复上述步骤直到机器狗能够直线行走

## 注意事项

- **电源**：确保使用足够功率的电源，SG90舵机在运动时电流较大
- **机械结构**：确保机械结构牢固，避免舵机负载过大
- **校准**：新组装的机器狗需要进行仔细的校准
- **安全**：在机器狗运动时，保持安全距离，避免被舵机夹伤

## 故障排除

1. **舵机不转动**：
   - 检查电源连接
   - 检查舵机引脚连接
   - 检查舵机是否损坏

2. **机器狗不平衡**：
   - 重新校准舵机偏移量
   - 检查机械结构是否对称

3. **语音控制无响应**：
   - 检查麦克风连接
   - 确保网络连接正常
   - 检查唤醒词设置
