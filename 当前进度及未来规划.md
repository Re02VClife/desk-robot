### 数据流

```
串口终端 > 你好
  → text_console 任务 (getchar 逐字读取, 回车发送)
  → Application::SendTextInput("你好")
  → Protocol::SendTextInput → cJSON {"type":"text","text":"你好"}
  → MQTT/WebSocket → OpenClaw
  → TextInputMessageHandler → enqueue_asr_report + startToChat
  → LLM → TTS → ESP32 扬声器 + 屏幕显示回复
```

### 使用方式

烧录固件后，打开串口监视器：

```
文字控制台已启动，输入文字后按回车发送（/help 帮助 /quit 退出）
> 今天天气怎么样
[屏幕显示"user: 今天天气怎么样"]
[等待服务器回复...]
[屏幕显示"assistant: 今天天气晴..." + 喇叭播放语音]
```

   $PY -m pip install "torch==2.6.0+cu124" "torchvision==0.21.0+cu124" \
       -f https://mirrors.aliyun.com/pytorch-wheels/cu124/ -i https://mirrors.aliyun.com/pypi/simple/
   $PY -m pip install mujoco gymnasium -i https://mirrors.aliyun.com/pypi/simple/
   $PY -m pip install "lerobot==0.4.4" -i https://mirrors.aliyun.com/pypi/simple/   # 钉版本，不加 extra
   # lerobot 会把 torch 拉成最新 CPU 版，需强制装回 CUDA 版
   $PY -m pip install --force-reinstall --no-deps "torch==2.6.0+cu124" "torchvision==0.21.0+cu124" \
       -f https://mirrors.aliyun.com/pytorch-wheels/cu124/
   # smolvla 运行依赖（num2words 在 Aliyun 镜像缺失 → 用清华镜像）
   $PY -m pip install "transformers>=4.57.1,<5.0.0" "num2words>=0.5.14,<0.6.0" \
       -i https://pypi.tuna.tsinghua.edu.cn/simple/

### 10.3 机械臂驱动板调试（Bus Servo Adapter V1.1）

#### 关键发现

| 发现 | 详情 |
|------|------|
| 丝印 `G T R V V` | G=GND, T=TX, R=RX, V=3.3V输出 + 5V输出(需USB) |
| 跳线帽 2 针 | 短接=USB模式, 断开=UART模式 |
| TX/RX 接法 | 标准交叉（ESP32 TX→板子R, ESP32 RX→板子T） |
| VCC 必须接 | 3.3V 脚必须接 ESP32 3.3V，否则 UART 电路不工作 |

#### 接线（商家确认）

```
ESP32          驱动板
 TX  (IO9)  →   R
 RX  (IO10) ←   T
 3.3V       →   V (3.12V 脚)
 GND        →   G
```
